📋 설치 방법

  1. 다운로드된 ZIP 파일을 우클릭하여 "압축 풀기" 선택
  2. 압축 해제된 폴더에서 ClockApp.exe 실행
  3. Windows 보안 경고가 나오면 "추가 정보" → "실행" 클릭
  4. 건강한 업무 시작! 🚀

시스템 요구사항

  • Windows 10 이상
  • RAM 2GB 이상 (가벼운 앱!)
  • 디스크 공간 30MB
  • 인터넷 연결 (건강 통계 동기화)
